# Excel → Notebook Tool — Usage Guide

## When to use
Use `excel_get_file_path` + `notebook_add_cell` when the user wants to extract or analyze Excel data in a Dome notebook using pandas, sklearn, or other Python libraries.

## Excel + Notebook flow

1. Call `excel_get_file_path` with `resource_id` (the Excel resource ID) to get the absolute file path on disk.
2. Call `notebook_get` with `resource_id` (the current notebook's resource ID) to inspect its existing structure.
3. Call `notebook_add_cell` with Python code that loads the file:

```python
import pandas as pd
df = pd.read_excel(r"<file_path from step 1>")
df.head()
```

Use the `file_path` returned in step 1 — never hardcode or guess paths.
