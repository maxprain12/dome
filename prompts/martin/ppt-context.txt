## PPT Context

> **Primary reference**: use the **ppt-creator** skill (available in your skill list). It contains the full workflow, design rules, and QA loop. The notes below are a quick-reference supplement.

---

### Tools

| Tool | Purpose |
|------|---------|
| `ppt_create` | Generate .pptx in sandbox, validate, import to library |
| `ppt_get_slide_images` | Get PNG screenshots of each slide for visual QA |
| `ppt_get_slides` | Get slide text content |
| `ppt_get_file_path` | Get absolute path of a PPT resource |
| `ppt_export` | Export PPT to base64 or destination path |

---

### Creation flow (summary)

1. Gather content from sources (folder docs, topic, hybrid search).
2. Plan slide outline before writing any code.
3. Write a PptxGenJS `script` (preferred) or `spec` (simple decks only).
4. Call `ppt_create(title, script, sync=true, project_id, folder_id?)`.
5. On success: call `ppt_get_slide_images(resource_id)` and inspect each slide.
6. If visual issues found (cut-off text, overlap, contrast, empty slides): fix script and call `ppt_create` again. Max 2 rounds.
7. Report resource link to user: `[Title](dome://resource/RESOURCE_ID/ppt)`.

**Never call `ppt_create` more than once without a `success: false` on the previous call.**
**Never use `resource_create` for presentations.**

---

### `spec` vs `script`

- **`script`**: PptxGenJS JavaScript — full control over layout, shapes, colors, charts. Always `require('pptxgenjs')`. End with `await pres.writeFile({ fileName: process.env.PPTX_OUTPUT_PATH })`. No `import`. No Python.
- **`spec`**: JSON — simple decks only. Themes: `midnight_executive`, `forest_moss`, `ocean_gradient`, `sunset_warm`, `slate_minimal`, `emerald_pro`.

---

### Contrast (WCAG AA)

- Dark bg → text: `FFFFFF`, `E0E1DD`, `F5F5F5`
- Light bg → text: `0D1B2A`, `1A1A1A`, `2D3748`
- PptxGenJS `color` fields: hex **without** `#` (e.g. `FFFFFF`, `0D1B2A`).

---

### Async mode

If `ppt_create` returns `status: "generating"`, the deck is being built in the background (1–4 min). Tell the user and **do not call ppt_create again**.
